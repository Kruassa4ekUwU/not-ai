# not-ai 🧠

Телеграм бот с собственной моделью на базе Markov chain 3-го порядка.
Учится на сообщениях чата, отвечает когда его тегают, ищет инфу в интернете.

## Запуск

### Переменные окружения
```
BOT_TOKEN=твой_токен_бота
```

### Локально
```bash
pip install -r requirements.txt
python main.py
```

### Railway
1. Залей репо на GitHub
2. Создай проект на Railway → Deploy from GitHub
3. Добавь переменную `BOT_TOKEN` в Variables
4. Готово

## Команды
- `/start` — приветствие
- `/stats` — статистика (сообщений в базе, слов в модели)
- `@bot вопрос` — получить ответ / поискать в интернете
